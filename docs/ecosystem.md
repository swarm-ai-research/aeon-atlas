## Matching method

Each ECOSYSTEM.md project name is tokenized (lowercased, alphanumeric only) and scored against every fork's owner name, repo name, description, and the X handle in the project's link cell. The owner-name match and the X-handle match each contribute up to 3 points; the repo-name match contributes 2; a description match contributes 1. Any fork scoring ≥ 2 is considered a match (best score wins). False positives and missed matches can be fixed by editing the helper in `scripts/atlas.mjs:matchEcosystemToFork`.
