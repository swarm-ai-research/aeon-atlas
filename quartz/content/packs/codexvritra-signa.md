# signa

The full SIGNA skill suite — wallet-signed cross-platform agent messaging, multi-agent coordination (broadcast and delegate across the network), plus integrations with Aeon ecosystem partners already on Base: Bankr resolver and recent launches, gitlawb activity for any wallet, MiroShark sim stats and wallet-signed sim fire, plus x402 receipts and bounded spend mandates (a human grants a signed budget, the agent spends within it and asks for more). Lives at aeon-skills/ inside the SIGNA monorepo so the wire format, SDKs, and skill pack all version together.
