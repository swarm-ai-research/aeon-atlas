## Layer 4 — Discovery infrastructure

In-repo mechanism for things outside the project to be found from inside it. Registry, installer, catalog feed.

- [skill-packs.json](https://github.com/aaronjmars/aeon/blob/main/skill-packs.json) — Machine-readable registry of community packs. The single source of truth Layer 3 renders from.
- [install-skill-pack (CLI)](https://github.com/aaronjmars/aeon/blob/main/install-skill-pack) — On-demand pack installer. Runs a security scan against every SKILL.md and prompts on HIGH findings before writing a file.
- [sparkleware-catalog (skill)](https://github.com/aaronjmars/aeon/tree/main/skills/sparkleware-catalog) — Joins the registry to live GitHub signals and writes skill-packs-catalog.json — the machine-readable feed Layer 5 consumes.

## Layer 5 — Storefronts

## Layer 6 — Audit

Tools that don't ship code — they ship receipts against the code. Run consensus benches against PRs, file structured findings, hold the kernel accountable to its own standards.

- [AntFleet](https://www.antfleet.dev) — Two-model consensus PR review (Claude Opus + GPT-5). Findings filed under aeon Issue #184 with file-and-line citations.

## Layer 7 — Intake

Inbound funnel: how outside contributors land work in the kernel safely. Pre-flight scanning, secret-handling conventions, structured triage.

- [pr-skill-triage (skill)](https://github.com/aaronjmars/aeon/tree/main/skills/pr-skill-triage) — Workflow-dispatch skill that posts a structured comment on every inbound SKILL.md PR — security scan, required secrets, cron-slot conflicts.
- [prefetch / postprocess sandbox shims](https://github.com/aaronjmars/aeon/tree/main/scripts) — Maintainer lands scripts/prefetch-*.sh + scripts/postprocess-*.sh ahead of community SKILL.md PRs so contributors never carry secrets in their own commits.

## The blank spaces

Categories Hermes Atlas has and Aeon doesn't — yet. Each one is a slot waiting for a single actor's contribution.

- **Workspaces & GUIs** — No third-party operator console yet.
- **Multi-Agent Orchestration** — No skills-of-skills composition framework yet.
- **Guides & Documentation** — No community tutorial site yet.
